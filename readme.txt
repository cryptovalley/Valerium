This is the repository for Valerium and it contains the Sourcecode and Wallets.
Valerium is a SHA-256 blockchain coin that is for the people. As such there will be no Initial Coin Offering (ICO) for this coin.
This coin is built to mine. There is no MAC wallet available at this time. There are only Windows and Linux wallets.
Instructions on how to mine the coin will be added in the near future.
